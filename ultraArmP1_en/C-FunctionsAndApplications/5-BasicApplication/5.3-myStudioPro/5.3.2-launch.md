## 1 Connection Mode Switching

The software defaults to `Wired` connection mode. The software supports two connection modes: `Wired` connection and `Wireless` connection. You can choose the connection method for the robot by switching between these modes.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/launch-2.png" />

When clicking the switch button, a secondary confirmation popup appears. The popup contains specific operation instructions -- read carefully and proceed, then click the confirm button to switch.

- If currently in wired connection: clicking this button shows the wireless connection confirmation popup
- If currently in wireless connection: clicking this button shows the wired connection confirmation popup

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/launch-3.png" />

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/launch-4.png" />

Note: Wireless connection only supports `IPv4` connections.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/launch-7.png" />

## 4 Login Module

### Wired Connection Login

Automatically scans all available serial port devices on the system, with scanned port numbers sorted in ascending order. Supports memory function -- when the page loads again, the last successfully used port number is selected by default. Select the correct serial port and click the login button to enter the software main page.

### Wireless Connection Login

Validates IP address input, only allowing legitimate IPv4 addresses. The specific login IP needs to be viewed through the MiniRobot screen.

After confirming the login information is correct, click the `Login` button to perform the login operation. Once the software successfully establishes communication with the robot, you can enter the software's main page and use all functional modules within the software.
