## 3 Pin Configuration

Click the `Pin Configuration` icon button. This module is used to configure and monitor the digital IO interfaces of the robotic arm base and the end tool. By combining the physical simulation view with the logical configuration table, users can define the signal direction (input/output) and default level status (high level/low level) of the pins.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-5.png" />

**Default level**

Configurable pin default levels: In output mode, the physical level is sent; in input mode, the level state is configured via pull-up/pull-down resistors (used for testing when no external connection is present). By system default, input pins are configured with internal pull-up (high level), and output pins initially output low level.

**Pin configuration**

Configurable input and output for each IO port (except for IO1 and IO2 of the tool IO, which are fixed as input pins, and IO3 and IO4, which are fixed as output pins)

The pins 1 to 5 on the base are set as input by default; the pins 6 to 10 on the base are set as output by default.

After modifying the configuration of a single pin (input/output mode or default level), the configuration of that pin is updated in real time and synchronized to the corresponding pin configuration item in the [Debug Panel](./5.3.4-debugPlane.md)

> Note: The pin configuration page is dedicated to initializing default levels. Modifications to I/O ports will be synchronized to the Debug Panel in real time. The Debug Panel displays the actual current level status of the pins.

> Operate using the bottom IO pin number 1 as an example

The pin configuration has been changed from input to output. Once the modification is successful, the message 【The Command was issued successfully】 will be returned.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-6.png" />

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-7.png" />

The default level is changed from a high level to a low level. After the modification is successful, the message 【The Command was issued successfully】 will be returned, and the high-level indicator in the default status bar will disappear.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-8.png" />

After the modification is completed, go to the debugging panel page to view the modified items. The status has been synchronized.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-9.png" />

## 4 Update Check 

Clicking the `Update Check` icon will take you to the update status page. This page allows you to download, delete, and upgrade software drivers (myStudio Pro, screen driver, motion firmware).

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-10.png" />

When the software/firmware has the latest version and no upgrade operation has been performed, a red number 1 will be displayed next to the `Update Check` icon. This indicates that there is a new version pending update. You can then access this page to view the specific upgrade details.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-11.png" />

On this page, you can enable the automatic download function for the software or firmware. As the name suggests, the automatic download will silently download the latest version of the software or firmware in the background.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-12.png" />

When there is a new version of the software/firmware available, a pop-up window with upgrade information will appear when you enter the main page. Through this pop-up window, you can view the contents of the new version. If you click `Update later` or close the pop-up window, you will not proceed to the page for checking updates. On the contrary, if you click `Go to Update`, it will redirect you to the page for checking updates, but you still need to manually update.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-13.png" />

### myStudio Pro Upgrade

**Version Number**

By default, the current version of the software is displayed. If the current version is the latest one, it will show `Latest`; otherwise, it will show `Update`.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-14.png" />

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-15.png" />

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-16.png" />

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-17.png" />

Clicking the "Upgrade" button will initiate the download process of the installation package, and the download progress will be displayed at the same time.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-18.png" />

When the installation package is downloaded completely, the "Install" button will be displayed. Clicking on it will allow you to perform the software installation operation.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-19.png" />

## 5 Initial Position

Click the `Initial Position` icon button, and you will enter the initial position page. This page contains the `Initial Position` and `Packaging posture` contents. You can switch the content to be accessed by using the toggle buttons.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-41.png" />

Packing posture page: Displays the model diagram of the packing posture of the robotic arm, joint angle information, and the one click packing button. This button is a functional button. When this button is clicked, the robotic arm will move to the packing posture.

## 6 Test Module

Clicking the `Test Module` icon button will take you to the test page.

Test page: Mainly tests the communication connection of the robotic arm, firmware version, servo status, etc.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-44.png" />

Click the "Start Detection" button, and the test can begin.

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-45.png" />

After the test is completed, click the "Save Test Log" button. After each test item is completed, it will display whether the result of this test is passed or not. Green indicates normal, while any other color indicates abnormal. The test results can be saved to a local log file for viewing the specific test content.
