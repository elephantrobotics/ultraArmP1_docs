# Configuration Center

## 1 Interface Introduction

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/configuration-1.png" />

|Number|Function Introduction|
|--|--|
|1|Software Language Module, mainly for setting the language|
|2|Pin Configuration Module|
|3|Update Check Module|
|4|Initial Position Module|
|5|Test Module|
|6|Module Content Display Area|

## 2 Software Language

Click the `Software Language` icon button to enter the language settings page. Language selection supports `Chinese` and `English`.

## 3 Pin Configuration

Click the `Pin Configuration` icon button to configure and monitor the digital I0 interfaces of the robotic arm base and the end tool.

## 4 Check for Updates

Clicking the `Check for Updates` icon will take you to the update status page. This page allows you to download, delete, and upgrade software drivers (myStudio Pro, screen driver, motion firmware).

## 5 Initial Position

Clicking the `Initial Position` icon will take you to the Initial Position page. This page contains `Initial Position` and `Initial Position` content. You can switch between the content you want to access using the toggle button.

## 6 Test Module

Clicking the `Test Module` icon button will take you to the test page.
