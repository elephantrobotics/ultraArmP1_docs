# Q&A

**Q1: When update check page and attempting to download, it shows that the download failed and the file couldn't be downloaded properly?**

A: Before using the "Check for Updates" function, please make sure that your robotic arm is connected to the internet. Please note: What is required here is that the robotic arm itself is connected to the network, not the device accessing myStudio Pro. Without a network connection, the software driver cannot be downloaded properly. Additionally, network quality will also affect the function usage. A slower network connection may result in the failure of driver download. To ensure the normal operation of the function, please check the network status in advance. 

**Q2: Did the firmware upgrade fail during the process?**

A: Please ensure that the robotic arm is properly powered on during the upgrade process. Then try to upgrade again. If all upgrades fail, please contact the after-sales service for a solution.
