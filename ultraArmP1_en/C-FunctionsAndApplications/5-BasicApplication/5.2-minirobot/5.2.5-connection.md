# Connection

In the Program interface, select the Connection function with the asterisk, then press button C to enter the Connection function.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/program_Connection.png" width="300" align="center"/>

After entering the Connection function, you can select USB to view the machine's SN code and configure the serial communication baud rate.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/Connection_USB.png" width="300" align="center"/>

After selecting USB, it will display the current machine's SN code and the currently configured serial baud rate.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/USB.png" width="300" align="center"/>

Select Baud rate to modify the baud rate configuration (select the corresponding baud rate, press button C to confirm the change).

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/Baud_rate.png" width="300" align="center"/>

Return to the Connection function, you can select WLAN to view the machine's WiFi connection information and select WiFi to connect, etc.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/Connection_WLAN.png" width="300" align="center"/>

After selecting WLAN, it will display the current machine's WiFi connection status. Press button C to enter the configuration page, which supports switching, searching nearby WiFi and connecting, historical connection records (supports direct connection to previously connected WiFi, and can also delete historical connection records), and viewing the current machine's MAC.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/WLAN.png" width="900" align="center"/>

After entering the WLAN configuration page, you can select the Switch option and press button C to turn WiFi on/off. After turning off WiFi, the Available and History options will be grayed out and cannot be clicked.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/WLAN_Switch.png" width="600" align="center"/>

After turning on WiFi and selecting Available to enter, it will search nearby WiFi and display them.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/Available_Scanning.png" width="600" align="center"/>

After selecting the WiFi to connect and pressing button C, connection will start (the following four images respectively show: connecting, no password, connection failed, and connection successful).

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/Available_Connect.png" width="1200" align="center"/>

Return to the WLAN menu page, select the History page to enter. You can see historical connection record WiFi. Select the historical WiFi to connect and press button C to directly connect or delete this historical record WiFi.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/WLAN_History.png" width="600" align="center"/>

After selecting a historical connection WiFi and pressing button C to enter, you can again select connection to directly connect.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/WLAN_History_Connect.png" width="1200" align="center"/>

You can also select to delete this historical record.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/WLAN_History_Delete.png" width="900" align="center"/>

Return to the WLAN menu page, select properties to enter. You can see the current machine's MAC.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/WLAN_Properties.png" width="600" align="center"/>

Return to the Connection menu page, select BlueTooth to enter and view the Bluetooth connection status.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/Connection_BlueTooth.png" width="300" align="center"/>

After entering the BlueTooth page, you can see the current Bluetooth connection status.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/Connection_BlueTooth_status.png" width="900" align="center"/>

Press button C on the BlueTooth page to enter Bluetooth on/off.

<img src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.2-minirobot/BlueTooth_Switch.png" width="600" align="center"/>
