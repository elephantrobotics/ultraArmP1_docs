# Settings

In the Program interface, select the Settings function with the asterisk, then press button C to enter Settings function.

<img src="/resources/picture/screen_ui_images/program_Settings.png" width="300" align="center"/>


After entering Settings function, you can select Clear Error or View Log of python scripts running on the screen.

<img src="/resources/picture/screen_ui_images/Settings_ClearError.png" width="300" align="center"/>

Enter Clear Error page. If the robotic arm is not in joint limit, coupling position, or has triggered collision, it will prompt "No Errors."

<img src="/resources/picture/screen_ui_images/ClearError_NoError.png" width="300" align="center"/>

If there is a joint beyond the limit, it will prompt being in limit. Press button C to home to clear the limit error.

<img src="/resources/picture/screen_ui_images/ClearError_LimitError.png" width="300" align="center"/>


During homing, white text prompts "Backing..."

<img src="/resources/picture/screen_ui_images/ClearError_Backing.png" width="300" align="center"/>


After homing is complete, green text prompts "Finished!", and automatically returns to settings page after 3 seconds.

<img src="/resources/picture/screen_ui_images/ClearError_ResetComplete.png" width="300" align="center"/>

If the machine is in coupling position or has triggered collision, it will prompt being in coupling position or has collided. Press button C to clear the error.

<img src="/resources/picture/screen_ui_images/ClearError_CouplingError.png" width="300" align="center"/>
<img src="/resources/picture/screen_ui_images/ClearError_CollisionError.png" width="300" align="center"/>

During clearing, white text prompts "Clearing..."

<img src="/resources/picture/screen_ui_images/ClearError_Clearing.png" width="300" align="center"/>

After clearing coupling is complete, green text prompts "Cleared!", and automatically returns to settings page after 3 seconds.

<img src="/resources/picture/screen_ui_images/ClearError_Cleared.png" width="300" align="center"/>

If clearing coupling fails, red text prompts "failed!", and automatically returns to settings page after 3 seconds.

<img src="/resources/picture/screen_ui_images/ClearError_Failed.png" width="300" align="center"/>

Return to Settings page and select View Log to enter, you can view log files generated by python scripts running on the screen.

<img src="/resources/picture/screen_ui_images/Settings_ViewLog.png" width="300" align="center"/>

After entering View Log, if there are no log files, it will prompt "No Log."

<img src="/resources/picture/screen_ui_images/ViewLog_NoLog.png" width="300" align="center"/>

If there are log files, it will load and display all log files.

<img src="/resources/picture/screen_ui_images/ViewLog.png" width="600" align="center"/>

You can select the log file you want to view and press button C to enter and view detailed logs of this file. You can press button C again to enter delete page, you can delete this file.

<img src="/resources/picture/screen_ui_images/ViewLog_LogData.png" width="600" align="center"/>

After entering delete page, press button C to prompt whether to confirm deleting this log file. After confirmation, it will be deleted.

<img src="/resources/picture/screen_ui_images/ViewLog_Log_Delete.png" width="1200" align="center"/>
