# 5.4 零位校准

## 方式一：MiniRobot 本地校准（推荐）

ultraArm P1 支持通过 MiniRobot 本地系统直接进行零位校准，无需连接电脑。

### 操作步骤：

1. 在主界面按 A 键进入菜单
2. 选择 Calibration（校准）功能
3. 选择 Manual（手动校准）进入
4. 通过 A、B 键选择需要校准的关节（被选中的关节区域会高亮反转）
5. 使用 A、B 键调整关节位置，或将关节手动拖动至物理零位刻度线
6. 按下 C 键保存校准位置
7. 对应关节将自动执行校准动作，完成后角度数据变为 0°

> 短按 A/B 键时步距角为 0.1°，各关节对准机械臂上的物理零位刻度线即可。
