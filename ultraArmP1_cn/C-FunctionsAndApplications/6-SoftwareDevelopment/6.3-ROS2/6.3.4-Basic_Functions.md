## mycobot_ros2安装

参考 [ultraArm P1 mycobot_ros2安装](./6.3.1-EnvironmentSetup.md)

执行下面命令安装 ultraArm P1 ROS2 代码：

```bash
$ cd ~/colcon_ws/src
# For the humble branch
$ git clone -b p1-humble https://github.com/elephantrobotics/mycobot_ros2.git
# For the foxy branch
$ git clone -b p1 https://github.com/elephantrobotics/mycobot_ros2.git
$ cd ~/colcon_ws
$ colcon build
$ source ~/colcon_ws/install/setup.bash
$ sudo echo 'source ~/colcon_ws/install/setup.bash' >> ~/.bashrc
```

# 控制机械臂

>> **注意：pymycobot 驱动库的版本必须大于4.0.5， Moveit2仅支持 ROS2 Humble版本，请使用对应分支的代码运行**

## 使用前准备

在使用案例功能之前，请先确认以下硬件和环境准备齐全：

- **硬件设备**  
  - ultraArm P1 机械臂
  - 电源适配器  
  - USB Type-C串口线

- **软件与环境**  
  - PC虚拟机 ubuntu 20.04系统
  - 已安装 Python 3.6 及以上版本  
  - 已安装 `pymycobot` 库（通过 `pip install pymycobot` 终端命令安装）  
  - 确保 ultraArm P1 已正确接通电源、串口线，并且虚拟机系统能检测到串口号。
  
  - **验证**：验证机械臂串口是否存在，终端输入：
  
    ```bash
    ls /dev/tty*
    ```

    如果输出的串口信息有 `/dev/ttyACM*` 或者 `/dev/ttyUSB*`，表示串口已被检测到。

  - **串口权限**： 如果串口存在，需要赋予可执行权限:

    ```bash
    sudo chmod 777 /dev/ttyUSB*
    # 或者
    sudo chmod 777 /dev/ttyACM*
    ```

打开命令行并运行：

```bash
# 默认串口名为"/dev/ttyUSB0"，波特率为1000000.
ros2 launch ultraarm_p1 slider_control.launch.py port:=/dev/ttyUSB0 baud:=1000000

```

它会**打开 rviz2 和一个滑块组件**，你会看到类似下面的内容：

<img src =../../../resources/C-FunctionsAndApplications/6-SoftwareDevelopment/6.3-ROS2/rviz2-3.png
width ="1200"  align = "center">

然后，您可以在 rviz2 中通过拖动滑块**来控制模型的移动**。真实的 ultraArm 机器人也会随之移动。

**请注意：由于在命令输入的同时机械臂会移动到模型目前的位置，在您使用命令之前请确保 rviz 中的模型没有出现穿模现象**

**不要在连接机械臂后做出快速拖动滑块的行为，防止机械臂损坏**

**现象说明：**

```text
ultraArm P1 在 MoveIt RViz 中进行规划预览时，J2/J3 部分联动外壳可能出现短暂视觉错位。该现象仅出现在 MoveIt 的目标状态或规划轨迹预览中，普通 RViz 模型显示和执行后的机器人状态正常。

原因是 ultraArm P1 含有平行连杆/联动机构，URDF 使用 mimic joint 进行近似建模，而 MoveIt 的规划轨迹显示对 mimic/闭链结构支持有限，预览状态下可能没有完整同步所有联动外壳。

J2/J3 的机械可行区域属于两个主动关节之间的组合约束，原生 demo.launch 不会自动限制该类非线性约束。当前在 sync_plan.py 中增加了执行前检查，只有 J2/J3 组合满足 valid_region(J2, J3) 时才会下发到真实机械臂。

该现象不影响 MoveIt 路径规划结果，不影响实际控制关节 J1/J2/J3/J4 的执行，也不影响真实机械臂运动。测试时以 Execute 后的当前机器人状态、sync_plan.py 的安全检查结果和真实机械臂运动为准。
```

打开命令行并运行：

```bash
ros2 launch ultraarm_p1_moveit2 demo.launch.py
```

### 修改运动速度

为了防止关节在实际机械臂运动过程中晃动，需要降低关节的运动速度。

- 修改机械臂 Python API 的速度参数，启动程序时增加速度参数，默认25。

```python
ros2 launch ultraarm_p1_moveit2_control sync_plan.launch.py port:=/dev/ttyUSB0 baud:=1000000 speed:=25
```

- 在 Moveit RViz 界面中，修改速度和加速度的缩放比例。在这里，将其改为 0.1，然后保存当前配置。`ultraarm_p1_moveit2_control` 功能包默认配置 0.5。
