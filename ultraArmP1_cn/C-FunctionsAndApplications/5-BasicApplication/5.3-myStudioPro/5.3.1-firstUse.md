# 首次使用

## myStudio Pro支持的操作系统如下

- Windows 10/11

- macOS

- Linux_x86_64

## 安装myStudio Pro

> 以下安装步骤以Windows操作系统为例，查看本章节内容讲解前，请保证您已拥有软件的安装包

1. 双击软件安装程序，开始软件安装，安装用户选项按您自身需要选择。

<img width = "550" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/download-1.png" />

2. 点击【下一步】，选择软件的安装路径，按您自身需要选择，建议安装至C盘以外的路径下

<img width = "550" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/download-2.png" />

3. 确认无误后，最后点击【安装】，耐心等待直至软件安装完成

<img width = "550" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/download-3.png" />
   
4. 点击【完成】，即可运行软件

## 卸载myStudio Pro

> 以下安装步骤以Windows操作系统且建立桌面快捷方式为例，查看本章节内容讲解前，请保证您的设备已成功安装软件

1. 选择软件，并鼠标右键。

<img width = "550" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/uninstall-1.png" />

2. 选择【打开文件所在的位置】，找到【Uninstall xxx.exe】执行文件鼠标双击。

<img width = "550" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/uninstall-2.png" />

3. 打开【解除安装向导】，点击下一步即可完成软件卸载

<img width = "550" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/uninstall-3.png" />
