## 写字画画

### 主页面

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-2.png" />

主界面是进行G代码文件管理、轨迹预览与作业执行的核心入口，主要划分为以下区域:

#### 顶部工具栏

提供返回、新增、案例库、快速移动、自由移动五个功能按钮。新增用于进入工作台创作图案;案例库内置预设G代码文件;快速移动支持点动调试与特殊点设置;自由移动可手动拖拽机械臂至目标点位

**返回:** 点击按钮可返回场景应用主页面。

**新增:** 点击按钮后，可进入写字画画工作创建编辑新项目。

**案例库:** 下拉菜单，可浏览并选用内置预设的可加工G代码文件，快速加载其加工轨迹至中间预览区。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-3.png" />

**快速移动:** 点击后弹出快速移动页面，支持设置工作原点、轨迹起点、精确点位调试以及实时坐标读取。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-4.png" />

**自由移动:** 点击后会弹出消息框，点击确认后，机械臂可手动拖拽，可移动机械臂至初始点位，简化调试过程。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-5.png" />

#### 左侧视图控制区

提供主视图(恢复默认视角)、向左旋转/向右旋转/向上旋转/向下旋转(调整观察角度)以及左上点/右上点(将视图中心快速定位至作业区域的左上角或右上角)，便于多维度检查加工轨迹。

**主视图:** 点击按钮后恢复3D预览区至默认初始视角。此页面主视图默认为俯视图。

**向左旋转:** 支持单击和长按，单击旋转15度，长按以30度/s的速度向左持续旋转，直至松开按钮停止。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-6.png" />

**向右旋转:** 支持单击和长按，单击旋转15度，长按以30度/s的速度向右持续旋转，直至松开按钮停止。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-7.png" />

**向上旋转:** 支持单击和长按，单击旋转15度，长按以30度/s的速度向上持续旋转，直至松开按钮停止。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-8.png" />

**向下旋转:** 支持单击和长按，单击旋转15度，长按以30度/s的速度向下持续旋转，直至松开按钮停止。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-9.png" />

**右上点:** 点击按钮后将视角中心快速定位至作业区域的右上点。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-10.png" />

**左上点:** 点击按钮后将视角中心快速定位至作业区域的左上点。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-11.png" />

#### 中间3D轨迹预览区

以三维形式展示当前选中G代码文件的加工轨迹，扇形边界代表机械臂可加工范围。右下角提供缩放控制，支持对视图进行精细调节。

鼠标可拖动视角平移，滚轮可决定放大放小的中心。

**校准:** 点击"校准"按钮，将视图恢复至默认主视角，缩放比例恢复至默认值(10mm/格)

**放大:** 点击"放大"按钮一次，视图放大一级，比例尺按预设序列向更小值切换(例如从10mm/格变为5mm/格)。支持连续点击逐步放大至最小比例尺1mm/格，达到最小比例后点击按钮无反应。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-12.png" />

**缩小:** 点击"缩小"按钮一次，视图缩小一级，比例尺按预设序列向更大值切换(例如从10mm/格变为50mm/格)。支持连续点击逐步缩小至最大比例尺100mm/格，达到最大比例后点击按钮无反应。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-13.png" />

**比例尺:** 在右下角显示当前比例尺，格式如"10mm"，表示当前网格每一小格所代表的实际长度。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-14.png" />

#### 文件管理与运行控制区 

按时间倒序展示G代码文件列表，支持重命名、导入、导出、删除等基础操作。底部为运行面板，显示当前文件的预计耗时、运行时间及运行状态;插入至Blockly可跳转至积木编程界面，便于构建批量加工流程。

**G代码文件列表:** 该区域展示G代码文件列表，列表项字段包含文件名、文件大小以及保存日期，列表项图片为加工图案。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-15.png" />

**重命名:** 选中列表项后，点击按钮后可重命名列表项的文件名。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-16.png" />

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-17.png" />

**导入:** 点击导入按钮后，可从本地导入G代码文件(仅支持.gcode, .nc, .cnc)。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-18.png" />

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-19.png" />

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-20.png" />

**导出:** 点击导出按钮后，可导出所选文件到本地。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-21.png" />

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-22.png" />

**删除:** 点击删除按钮后，弹出弹窗确认后即可删除所选文件，点击确认后删除。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-23.png" />

#### 运行面板

页面载入时无选中G代码文件，运行面板为隐藏状态。当选中G代码文件后，显示该G代码文件的运行面板，面板展示该文件的预计耗时、运行时间以及机械臂的当前状态，支持对加工任务进行运行、暂停、停止、继续等控制操作。

**长按至工作原点:** 长按按钮后机械臂运行至预设的工作原点。回原点完毕后，将展示运行、暂停、停止按钮。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-25.png" />

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-26.png" />

**运行:** 点击运行按钮后，机械臂开始运行加工文件。运行时间开始计时、当前状态变为运行中，运行按钮转变为禁用状态，暂停按钮和停止按钮转为启用状态。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-27.png" />

**暂停:** 点击暂停按钮后，机械臂暂停运行。面板的当前状态转为暂停中，运行按钮转变为恢复按钮，同时暂停按钮为禁用状态，恢复按钮和停止按钮为启用状态。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-28.png" />

**恢复:** 点击恢复按钮后，机械臂恢复运动。

**停止:** 点击停止按钮后，软件停止下发指令，当前状态为已停止，按钮状态除运行按钮为启用状态之外，暂停按钮和停止按钮皆为禁用状态。机械臂将已发送指令执行完毕后停止，运行面板的当前状态为待开始。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-29.png" />

**插入至Blockly:** 点击插入至blockly按钮后，会跳转至「积木编程工作台」，可通过积木编程实现批量运行。

### 工作台

写字画画工作台是进行图案创作、编辑、参数配置与作业执行的核心界面。

#### 顶部工具栏

项目操作与素材导入包含返回上级、打开本地项目、保存当前作品、调用内置模板库以及拍照上传图片，实现从外部资源获取到项目文件管理的全流程操作。点位工具提供快速移动与自由移动，快速移动点击后弹出调试弹窗，支持设置工作原点、轨迹起点及精确点位调试;自由移动模式支持直接拖拽机械臂至目标点，简化点位调试流程。

#### 左侧工具箱 

汇聚图形绘制与编辑核心工具，提供对象选择、画布平移、钢笔绘制、圆形绘制、文字添加、预设图形插入以及撤销与重做。其中撤销与重做支持快捷键操作(Ctrl+Z/Ctrl+Y)，便于快速回退或恢复编辑步骤。

**选择工具:** 用于选中画布中的图形对象。支持单击单选图形对象，或拖拽鼠标进行框选多选。选中对象后，图形对象颜色变为蓝色，支持进行移动、缩放、旋转等编辑操作。按住Shift键可连续加选多个对象。

**移动画布:** 用于拖拽平移画布视角，便于观察画布的不同区域。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-74.png" />

**钢笔工具:** 用于绘制自由曲线或贝塞尔路径。点击完成按钮可结束本次绘制。

**矩形工具:** 用于绘制矩形图形。在画布上按住鼠标左键并拖拽即可生成矩形，松开鼠标完成绘制。按住Shift键拖拽可绘制正方形。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-76.png" />

**圆形工具:** 用于绘制圆形或椭圆图形。在画布上按住鼠标左键并拖拽即可生成椭圆，松开鼠标完成绘制。按住Shift键拖拽可绘制正圆。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-77.png" />

**文字工具:** 用于在画布中添加文字对象。点击画布出现文字输入框，可输入中文或英文文本，输入完成后点击外部区域确认。生成的文字对象支持后续编辑文本内容、设置字体、字号等属性。

**重做:** 用于恢复已撤销的操作。仅在执行过撤销操作后可触发，支持连续重做直至恢复至最新状态。同时支持快捷键Ctrl+Y快速触发重做功能。

扇形区域为机械臂实际的可加工区域。鼠标右键可使用工具菜单。

**校准:** 点击按钮，将画布视图恢复至默认居中视角，即画布中心(0，0)。对齐视图中心，缩放比例恢复至默认值(10mm/格)。

#### 右侧属性台 

作为图形属性配置与加工准备的核心区域，通过页签切换编辑与处理两大功能模块。编辑模块支持图层管理(重命名、层级调整、删除)以及图形对象的变换属性调整(坐标、尺寸、文本以及图片处理模式);处理模块支持选中图形对象并创建工具路径，进入后续加工参数配置与G代码生成流程，实现从设计到加工的无缝衔接。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-85.png" />

**图层列表:** 展示当前画布中所有图形对象的图层列表。每个图层项显示对象名称。支持单选或多选选中图层，画布中对应对象颜色变为蓝色，同时底下变换区域同步显示该对象的属性参数(仅为单选时)。列表项左边有显示/隐藏按钮，支持显示或隐藏图层。

- 不勾选图层时，图层列表四个管理按钮皆为禁用状态
- 勾选一个图层时(图片类)，四个按钮皆为启用状态，显示变换区域和处理模式，支持修改对象属性(坐标、尺寸、是否旋转或镜像以及图片处理模式)
- 勾选一个图层时(图形类)，四个按钮皆为启用状态，显示变换区域，支持修改对象属性(坐标、尺寸、是否旋转或镜像)
- 勾选一个图层时(文字类)，四个按钮皆为启用状态，显示变换区域和文本处理，支持修改对象属性(坐标、尺寸、是否旋转或镜像以及文本处理)
- 勾选两个或两个以上图层时，只有"删除”按钮为启用状态，其余三个按钮皆为禁用状态

**重命名:** 单选图层后，点击按钮后可重命名图层名称。

**缩小:** 点击"缩小"按钮一次，视图缩小一级，比例尺按预设序列向更大值切换(例如从10mm/格变为50mm/格)。支持连续点击逐步缩小至最大比例尺100mm/格，达到最大比例后点击按钮无反应。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-57.png" />

**比例尺:** 在右下角显示当前比例尺，格式如"10mm"，表示当前网格每一小格所代表的实际长度。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-58.png" />

#### 右侧调试台

运行面板支持进行调试运行，若在调试运行中发现不当，可返回调整编辑，如若确认了加工文件可保存返回至主界面

**长按至工作原点:** 长按按钮后机械臂运行至预设的工作原点。回原点完毕后，将展示运行、暂停、停止按钮。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-59.png" />

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-60.png" />

**运行:** 点击运行按钮后，机械臂开始运行加工文件。运行时间开始计时、当前状态变为运行中，运行按钮转变为禁用状态，暂停按钮和停止按钮转为启用状态。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-61.png" />

**暂停:** 点击暂停按钮后，机械臂暂停运行。面板的当前状态转为暂停中，运行按钮转变为恢复按钮，同时暂停按钮为禁用状态，恢复按钮和停止按钮为启用状态。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-62.png" />

**恢复:** 点击恢复按钮后，机械臂恢复运动。

**停止:** 点击停止按钮后，软件停止下发指令，当前状态为已停止，按钮状态除运行按钮为启用状态之外，暂停按钮和停止按钮皆为禁用状态。机械臂将已发送指令执行完毕后停止，运行面板的当前状态为待开始。

<img width = "1200" align = "center" src="../../../resources/C-FunctionsAndApplications/5-BasicApplication/5.3-myStudioPro/scene-63.png" />

**返回编辑：** 点击"返回"可返回「写字画画工作台」的编辑面板。

**保存并返回至主页面：** 保存并返回至主界面:点击按钮后跳转「写字画画主界面」，且G代码文件自动添加至主界面的文件列表开头并选中。
